About Me

I was first introduced to the world of programming in sixth form, where my friend would host a weekly coding club which I would attend after school. I was taught Python as my very first language and was immediately hooked in after my first “Print(“Hello, World”) statement.

I would attempt to develop small text-based adventure games during this time with the skills and knowledge I had learned but it wasn't until my Mathematics degree at University where I would truly find my passion for software development.

Throughout my course there would be several modules which involved us learning a programming language so that we could utilise its capabilities for solving mathematical and statistical problems. In this time I was taught MatLab, R, Maple, Bash and even Python again - as well as other important skills such as parallelising my code and externally running my program through the University's supercomputer.

After my degree I would use Codecademy to further develop my skills in my own time, first learning Java and its object-based principles and then transferring that knowledge to C# which is the language I'm now most familiar with.
